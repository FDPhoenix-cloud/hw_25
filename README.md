# Магазин товаров Rick and Morty (ДЗ №27)

Интерактивный магазин с поиском и фильтрацией товаров.

## Функциональность

- ✅ Живой поиск по названию и описанию товаров
- ✅ Нечувствительный к регистру поиск
- ✅ Визуальная обратная связь (красная рамка при отсутствии результатов)
- ✅ Сообщение об ошибке для пустого результата

## Установка

npm install


## Запуск

npm run dev


Откройте http://localhost:5173 в браузере.

## Структура проекта

- `src/App.tsx` — главный компонент с логикой поиска
- `src/components/ProductCard.tsx` — карточка товара
- `src/components/StandardButton.tsx` — кнопка
- `src/types/index.ts` — типы TypeScript
- `src/data.ts` — данные товаров
