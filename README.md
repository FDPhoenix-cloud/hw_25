# Магазин товаров Rick and Morty (ДЗ №26)

Проект с компонентным подходом, использующий React, TypeScript, Bootstrap 5.

## Установка

npm install


## Запуск

npm run dev


## Структура

- `src/types/` — TypeScript-интерфейсы
- `src/components/StandardButton.tsx` — универсальный компонент кнопки
- `src/components/ProductCard.tsx` — компонент карточки с 4 кнопками
- `src/data.ts` — массив товаров

## Функциональность

Каждая карточка товара содержит 4 интерактивных кнопки:
- Добавить в корзину
- Посмотреть подробнее
- Кнопка + (увеличение количества)
- Кнопка - (уменьшение количества)

При клике на любую кнопку выводится сообщение в консоль браузера.
