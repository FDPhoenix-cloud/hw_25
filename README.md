# Галерея товаров Rick and Morty

Проект с галереей товаров на React + TypeScript + Vite с использованием Flexbox.

## Установка

npm install


## Запуск в режиме разработки

npm run dev


Откройте http://localhost:5173 в браузере.

## Описание

- Компонент `ProductCard` отображает одну карточку товара
- Галерея построена с использованием CSS Flexbox
- Каждая карточка имеет кнопку "Добавить в корзину" с выводом сообщения в консоль
